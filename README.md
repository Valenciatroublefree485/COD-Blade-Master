If you encounter issues while running the program, review these common fixes.

If the program closes unexpectedly, check your graphic drivers. Download the latest drivers from your GPU manufacturer website. Outdated drivers often cause compatibility problems with overlay software.

If the features do not activate, verify that you run the application as an administrator. Right-click the application icon and select "Run as administrator" to grant the necessary permissions.
